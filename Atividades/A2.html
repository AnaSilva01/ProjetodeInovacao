BMG Canvas

Business Model Canvas (Modelo de Negócio) é uma ferramenta de planejamento estratégico. Ele permite destrinchar uma ideia em várias partes, para que seja possível enxegar o modelo de negócio viável.

Foi criado por Alex Ostewalter e Yves Pigneur, o canvas é composto de nove blocos, nos quais se deve colocar os itens referentes a cada pilar do negócio. Ao final, se terá uma visão holística e flexível do modelo e poderá descobrir o que faz o negócio ser especial.

Por meio do Canvas pode-se refletir sobre o que pode ou não dar certo no negócio.

Project Canvas

O que vou fazer
Define qual será o "valor" (Razão ou motivo pelo qual as pessoas adquirem o produto e serviço.) do produto/serviço para o cliente.
Para quem?
Define quem são os clientes que o negócio pretende atender.
Canais
Define de que forma os produtos ou serviços chegarão até os clientes.
Relacionamento com clientes
Define os relacionamentos com clientes (como fará para conquistar e manter uma boa relação com os clientes, para ampliar as vendas e para que eles não troque o negócio ou serviço por outro).
Quanto e como receber
Define com será a entrada de dinheiro. (Quando e como os clientes pagarão pelo que é oferecido).
Como fazer
Define todos os recursos necessários para realizar a proposta de valor.(O que é preciso fazer para o negócio funcionar.)
Atividades principais
Relacionará as ações necessárias para a realização da proposta de valor.
Parcerias principais
Idetificará fornecedores e parceiros para apoiar a realização da proposta de valor.
Quanto gastar
Levantará o que vai gastar para a realização da proposta de valor. (todos os custos envolvidos para operação do negócio.)
